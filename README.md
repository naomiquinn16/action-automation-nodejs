## To get started

- run: 'npm install'

## To run a Tiny Tines story
- run: 'node app.js --story={story_name}'
- Example: To run sunset story: 'node app.js --story=tiny-tines-sunset.json'
- Example: To run today story: 'node app.js --story=tiny-tines-today.json'

## To run tests

- run: 'npm run test'
